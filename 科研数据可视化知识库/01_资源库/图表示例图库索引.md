## 3. 收录流程（给 Agent）

```text
1. 找到一张值得学的图（优先官方 gallery / 公开论文 figure / 教科书示例）
2. 在 07_示例图像库/网络优秀图表示例/ 建卡片（用 07 目录的模板）
3. 记录：原始链接 / 来源 / 作者机构 / license / 是否可下载
4. 提炼：架构、配色、字体、轴、图例、注释、多面板、可学习点
5. 写“可迁移规则”与“不建议照搬”
6. 双向链接回对应知识卡片第 9.1 节
```

## 4. license 分级处理

- **明确开放许可（CC/BSD/MIT 等）**：可下载缩略图存到 `07_示例图像库/网络优秀图表示例/参考图例_开源/` 并注明来源。
  - ✅ 已落地（2026-06-10）：下载了 scikit-learn / seaborn / matplotlib 官方 gallery 共 11 张（BSD / Matplotlib License），清单见 [[_来源与许可]]，并已内嵌对应示例卡片。
- **版权受限 / license 不明**：**只记链接 + 文字描述**，不保存图片（prompt 第八节硬要求）。
  - 已按此处理：期刊图（NEJM/Lancet/Cochrane）、R Graph Gallery、from Data to Viz、dotwhisker。
- 顶刊文章图：一律按“受限”处理，只登记链接 + 设计点。
