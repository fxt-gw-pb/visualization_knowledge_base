# 图表返修规则总表

> “症状 → 诊断 → 处方”的总表。Agent 自动返修时优先查这里；每张卡片第 11 节是该图**专属**返修规则的细化。
> 设计成 `IF 条件 THEN 动作`，便于未来写进作图 Skill 的后处理器。

## 1. 通用返修规则（适用所有图）

| 触发条件 (IF) | 返修动作 (THEN) |
|---|---|
| 坐标轴标签重叠 | 旋转 30–45°；或缩写/换行标签；或改为横向图（`coord_flip` / 交换 x,y）|
| 图例过长 / 占地过大 | 移到底部并分多列；或直接在图内标注（direct labeling，`ggrepel` / `ax.annotate`）|
| 颜色类别 > 8 | 分面；或合并低频类别为 “Other”；或叠加形状/线型辅助区分 |
| 颜色非色盲友好 | 换 Okabe-Ito / viridis / ColorBrewer-safe（见 [[色盲友好与打印友好原则]]）|
| 散点重叠严重（overplotting）| 降 alpha；jitter；hexbin / 2D density；或抽样 |
| 字体不一致 | 全图统一一种无衬线字体，统一基准字号 |
| 图太拥挤 | 增大画布尺寸；或拆成多面板（见 [[分面与多面板设计]]）|
| 装饰过多 | 去 3D / 阴影 / 渐变 / 冗余网格 / 重边框（极简主题）|
| 用于论文 | 导出 PDF/SVG，嵌字体，尺寸按单栏/双栏 |
| 用于汇报 | 同时导出 PNG(≥300dpi)+PDF，放大字号，提高对比度 |

## 2. 统计表达类返修

| IF | THEN |
|---|---|
| 误差棒未说明是 CI/SE/SD | 在图注或图例注明；统一全文口径 |
| p 值过多、堆满图面 | 改为全局检验（如 ANOVA/Kruskal）+ 只标**预设**的关键两两比较 |
| p 值格式不一 | 统一：`p < 0.001`，其余两位有效数字；避免 `p = 0.0000` |
| 缺参考线 | OR/HR 图加 x=1；差值/效应图加 0；校准图加 y=x；ROC 加对角线 |
| 对数关系画在线性轴上很挤 | 改 log scale 并明确标注，刻度用真实值 |
| 截断 y 轴误导（柱状图）| 柱状图必须从 0 开始；需放大差异改 point-range/dot plot |

## 3. 按图类型的专项返修（速查）

| 图类型 | 典型症状 → 处方 |
|---|---|
| 箱线图 | 样本量 < ~30 → 叠加原始点；组 > 8 → 横向或分面；强偏态 → log 或换 violin/raincloud。详见 [[箱线图_Boxplot]] |
| 小提琴图 | 样本量小密度不可靠 → 退回 box+jitter；尾部失真 → `trim`/限制带宽。详见 [[小提琴图_Violin]] |
| 雨云图 | 元素挤 → 拉开 box/point/density 偏移；点太多 → 抽样或 sina。详见 [[雨云图_Raincloud]] |
| 散点图 | overplot → alpha/hexbin；想表达趋势 → 加回归/LOESS + CI 带。详见 [[散点图_Scatter]] |
| 折线图 | 线太多成“意大利面” → 高亮重点+灰化其余/分面；个体轨迹乱 → 叠均值±CI。详见 [[折线趋势图_Line]] |
| 点估计+CI | 类别多挤 → 横向 + 按效应排序；多模型 → dodge/分面。详见 [[点估计置信区间图_PointRange]] |
| 森林图 | 个别 CI 极宽压缩主体 → 截断坐标轴并标注/log 轴；行多 → 分组小标题。详见 [[森林图_ForestPlot]] |
| KM 曲线 | 末端样本少抖动大 → risk table + 截断随访时间；曲线挤 → 限组数。详见 [[KM生存曲线_KaplanMeier]] |
| ROC | 多模型线缠绕 → 标 AUC+CI、限 ≤4 条/分面；类别不平衡 → 补 PR 曲线。详见 [[ROC曲线_ROC]] |
| 校准曲线 | 分箱抖动 → 改 loess 平滑校准；尾部稀疏 → 加 rug/直方图显示分布。详见 [[校准曲线_Calibration]] |
| 热图 | 标签过密 → 聚类 + 只标 top features / 隐藏部分；色阶被离群值拉爆 → 截断/分位映射/z-score。详见 [[热图_Heatmap]] |
| 多面板 | 子图字号不一 → 统一基准；共享图例重复 → 合并到一处；对齐差 → 用 patchwork/cowplot 对齐轴。详见 [[多面板组合图_MultiPanel]] |

## 4. 返修工作流（Agent 用）

```text
出图
 ↓
跑 [[科研图表质量检查清单]]
 ↓
对每个未通过项：
   查本表 IF→THEN，取处方
 ↓
应用处方（改代码参数，不改数据语义）
 ↓
重画 → 复查
 ↓
记录到卡片第 14 节“迭代记录”（第几版改了什么）
 ↓
全过 → 更新 status
```

## 5. 返修的红线（不允许的“返修”）

- ❌ 为了“好看”而**截断柱状图 y 轴**制造虚假差异。
- ❌ 反转发散色板让“坏”显得像“好”（见 [[发散变量配色]]）。
- ❌ 删除不利的离群点/不利数据点来“清爽”。
- ❌ 用平滑/插值制造数据里不存在的趋势。
- ❌ 把显著性阈值事后从 0.05 挪到 0.1 来“通过”。

返修只改**表达**，不改**事实**。
