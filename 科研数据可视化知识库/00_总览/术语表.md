# 术语表

> 本库反复出现的术语统一定义，避免 R/Python/统计/可视化各说各话。按主题分组。

## 图形语法（Grammar of Graphics）

| 术语 | 含义 |
|---|---|
| Grammar of Graphics | Wilkinson 提出、ggplot2/plotnine 实现的图形语法：图 = 数据 + 映射(aes) + 几何(geom) + 标度(scale) + 分面(facet) + 主题(theme) |
| aesthetic mapping (aes) | 把数据列映射到视觉通道（x/y/color/fill/size/shape）|
| geom | 几何对象：点、线、箱、面等（`geom_point`/`geom_boxplot`）|
| stat | 统计变换：在画之前对数据做的计算（如 bin、density、smooth）|
| scale | 标度：把数据值转成视觉值（颜色、坐标位置、大小）|
| facet | 分面：按变量把数据拆到多个子图（小多图，small multiples）|
| theme | 主题：非数据元素的外观（字体、背景、网格、刻度）|
| layer | 图层：geom+stat 的一次叠加，可多层堆叠 |

## 统计与误差度量

| 术语 | 含义 / 注意 |
|---|---|
| SD（标准差）| 描述**数据**离散程度；样本量变大不缩小 |
| SE（标准误）| 描述**统计量（如均值）**的抽样不确定性 = SD/√n；样本量变大缩小 |
| CI（置信区间）| 通常 95%；均值 CI ≈ mean ± 1.96·SE。**图里必须注明是哪种误差** |
| IQR（四分位距）| Q3−Q1，箱线图的箱体范围 |
| p 值格式 | `p < 0.001`；否则保留两位有效数字，如 `p = 0.023`（见 [[标签与注释设计]]）|
| OR（比值比）| odds ratio，logistic 回归；参考线 = 1，常用 log 轴 |
| HR（风险比）| hazard ratio，Cox 回归；参考线 = 1，常用 log 轴 |
| RR（相对危险度）| risk ratio；参考线 = 1 |
| effect size | 效应量；差值类参考线 = 0 |

## 颜色与配色

| 术语 | 含义 |
|---|---|
| sequential（顺序色）| 单向、亮度单调，编码连续量（低→高），如 viridis、Blues |
| diverging（发散色）| 双向、有中心值（常 0），编码正负/上下，如 RdBu |
| qualitative（分类色）| 离散、无序，编码类别，如 Okabe-Ito、Set2 |
| colorblind-safe | 色盲友好；约 8% 男性红绿色盲，避免红绿对立编码语义 |
| perceptually uniform | 感知均匀：相等数值差 → 相等视觉差（viridis、Crameri）|
| rainbow/jet | 彩虹/jet 色图，**避免**：亮度非单调、制造伪边界、打印灰阶失真 |
| Okabe-Ito | 8 色色盲友好分类调色板（事实标准）|
| ggsci | R 包，提供 NPG/Lancet/JAMA/NEJM/AAAS 等期刊配色 |

## 导出与尺寸

| 术语 | 含义 |
|---|---|
| 矢量图（PDF/SVG/EPS）| 用几何描述，无损缩放，适合线图/统计图 |
| 栅格图（PNG/TIFF/JPG）| 像素阵列，适合照片/热图密集像素；需关注 dpi |
| dpi | 每英寸点数；论文线图常需 600，含照片图 300 |
| 单栏图 (single column) | 约 85–90 mm 宽（≈3.3–3.5 in）|
| 双栏图 (double column) | 约 170–180 mm 宽（≈6.7–7 in）|
| bleed / bbox | 出血/边界框；导出时裁掉多余白边 |

## Framingham 相关

| 术语 | 含义（以本库真实数据为准）|
|---|---|
| 长格式 (long format) | 每人多行，一行一个随访期（本数据 PERIOD=1/2/3）|
| 基线 (baseline) | 第一次检查，PERIOD==1 |
| 右删失 (right-censored) | 研究结束仍存活/未发生事件，生存分析的删失类型 |
| TIMEDTH | 到死亡或删失的时间（天），配合 DEATH 做生存分析 |
