# 14 证据合成图族索引

> 面向**系统综述 / 荟萃分析(Meta-analysis)**：把多项研究的效应量合并成一个结论，并检查证据本身可不可信（异质性、发表偏倚）。生物统计与循证医学的核心产出图。

## 成员

| 图表 | 卡片 | 给什么信息 | 状态 |
|---|---|---|---|
| Meta 森林图 | [[Meta森林图_MetaForest]] | 各研究效应 + 权重 + 合并菱形 + I² | tested |
| 漏斗图 Funnel | [[漏斗图_Funnel]] | 效应 vs 精度，查发表偏倚/小研究效应 | tested |
| 累积/敏感性森林图 | ⬜ planned | 逐年累积证据、留一法 | planned |
| 气泡 Meta 回归图 | ⬜ planned | 效应随研究水平协变量变化 | planned |
| Galbraith / L'Abbé 图 | ⬜ planned | 异质性可视化 | planned |

## 选型速记

```text
合并各研究效应、看总体结论       → Meta 森林图（权重方块 + 合并菱形 + I²）
查发表偏倚 / 小研究效应          → 漏斗图（不对称 = 警惕）
看证据如何随时间累积 / 稳不稳    → 累积 / 留一法森林图（规划中）
解释异质性来源                   → 气泡 Meta 回归（规划中）
```

## 共性要点
- **效应量在对数尺度合并**（OR/RR/HR），CI 也在 log 尺度算，再指数化展示。
- 固定效应 vs 随机效应：异质性大(I² 高)时用随机效应(DerSimonian-Laird 等)。
- 漏斗图研究数 <10 时检验力低，慎下“无偏倚”结论。
- 与单研究的 [[森林图_ForestPlot]] 区别：Meta 森林图多了**权重、合并菱形、异质性统计量**。
