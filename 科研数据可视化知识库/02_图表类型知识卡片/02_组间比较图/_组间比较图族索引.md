# 02 组间比较图族索引

> 解决“**几组之间有没有差异**”。从“整组分布”到“仅估计 + 不确定性”按需选择。

## 成员

| 图表 | 卡片 | 给什么 | 状态 |
|---|---|---|---|
| 柱状图 Bar | [[柱状图_Bar]] | 计数/比例（必从 0）| tested |
| 点图 Dot plot | ⬜ planned | 离散点值 | planned |
| 点估计+CI Point-range | [[点估计置信区间图_PointRange]] | 估计 + 不确定性 | tested |
| 箱线+散点 | [[箱线图_Boxplot]] | 分布 + 原始点 | tested |
| 小提琴+箱线 | [[小提琴图_Violin]] | 形态 + 概括 | tested |
| 堆叠柱状图 Stacked bar | [[柱状图_Bar]] | 组成（绝对）| tested |
| 百分比堆叠 100% bar | [[柱状图_Bar]] | 组成（占比）| tested |

## 选型速记

```text
看整组分布差异        → 箱线/小提琴/雨云
看均值 + 不确定性     → 点估计+CI（point-range）
看计数/比例总量       → 柱状图（必从 0）
看类别组成            → 堆叠柱 / 百分比堆叠
强调原始数据透明      → 箱线+jitter / 雨云
```

## 反模式（重要）
- ❌ 用“柱状图 + 误差棒”表达连续变量均值（bar-bar plot）→ 隐藏分布，改 box/violin/point-range（[[图表返修规则总表]]）。
- ❌ 柱状图截断 y 轴制造差异。
