## 选型速记
```text
多变量效应并列        → 森林图（比值类 log 轴 + 参考线 1）
线性系数 β            → 系数图（参考线 0）
ML 模型“看什么”/可解释 → 特征重要性 / SHAP beeswarm
效应在不同人群是否一致 → 亚组分析森林图（看交互 p，不是层内 p）
连续协变量的非线性效应 → 边际效应图 / [[限制性立方样条_RCS]]
个体风险评分          → 列线图 Nomogram（搭配校准/区分度一起报）
```
