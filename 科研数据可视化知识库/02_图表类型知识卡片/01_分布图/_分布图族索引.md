# 01 分布图族索引

> 解决“**一个连续变量长什么样 / 组间分布如何**”。从“只给概括”到“给完整形态 + 原始点”逐级丰富。

## 成员

| 图表 | 卡片 | 给什么信息 | 状态 |
|---|---|---|---|
| 直方图 Histogram | [[直方图_Histogram]] | 单变量频数/密度（分箱）| tested |
| 密度图 Density | [[密度图_Density]] | 单变量平滑密度 | tested |
| 箱线图 Boxplot | [[箱线图_Boxplot]] | 五数概括 + 离群 | tested |
| 小提琴图 Violin | [[小提琴图_Violin]] | 完整密度形态 | tested |
| 雨云图 Raincloud | [[雨云图_Raincloud]] | 形态 + 概括 + 原始点 | tested |
| 山峦图 Ridgeline | [[山峦图_Ridgeline]] | 多组密度堆叠 | tested |

> 注：箱线图归在 [[图表类型索引]] 的“组间比较图”族目录，但分布信息也属本族；交叉引用。

## 选型速记

```text
只要单变量频数        → 直方图
要平滑分布            → 密度图
组间五数概括          → 箱线图
组间完整形态          → 小提琴图
形态 + 概括 + 原始点  → 雨云图（信息最全，首选）
很多组密度对比        → 山峦图
```

## 共性返修
- 样本量小 → 显示原始点（box+jitter / raincloud），别只给箱/violin。
- 偏态强 → log 轴（标注）或换能显形态的图。
- 详见 [[图表返修规则总表]]。
