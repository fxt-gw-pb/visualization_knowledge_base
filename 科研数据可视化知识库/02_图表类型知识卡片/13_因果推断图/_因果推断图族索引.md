# 13 因果推断图族索引

> 面向**真实世界研究(RWD/RWE)的因果推断**：观察性数据里要回答“处理 vs 对照到底有没有效应”，必须先证明**可比性**（混杂已调整）与**正性**（重叠充分），再报效应与剂量反应。本族图就是 PSM / IPTW / 标准化等流程的“合规证据图”。

## 成员

| 图表 | 卡片 | 给什么信息 | 状态 |
|---|---|---|---|
| 协变量平衡图 Love plot | [[协变量平衡图_LovePlot]] | 匹配/加权前后各协变量 \|SMD\|，证明可比性 | tested |
| 倾向评分重叠图 PS overlap | [[倾向评分重叠图_PSOverlap]] | 两组倾向评分分布重叠，证明正性/共同支持域 | tested |
| 限制性立方样条 RCS | [[限制性立方样条_RCS]] | 连续暴露与结局的非线性剂量反应（OR/HR + CI） | tested |
| 生存曲线（加权后） | ⬜ planned | IPTW 调整后的 KM/风险曲线 | planned |
| 爱情图+SMD 热图组合 | ⬜ planned | 多模型/多权重方案平衡对比 | planned |

## 选型速记

```text
证明两组基线可比（混杂已调平）  → 协变量平衡 Love plot（看 |SMD|<0.1）
证明两组可比性的前提：有重叠     → 倾向评分重叠图（无重叠 = 正性违背）
连续暴露不是线性的，要看形状      → 限制性立方样条 RCS（J/U/阈值效应）
报告调整后的处理效应             → 见 [[森林图_ForestPlot]]（ATE/ATT 的点估计+CI）
```

## 共性要点
- **因果三连**：先 PS 重叠（正性）→ 再平衡 Love plot（可交换性的可检验部分）→ 才报效应。任一缺失，效应估计都站不住。
- SMD 阈值惯例 **0.1**（<0.1 视为平衡）；优于用 p 值判平衡（p 值受样本量影响）。
- 这些图是**方法学合规**的一部分，常进正文或补充材料，审稿人必看。
- 与 [[缺失数据图_Missingness]] 联动：建 PS 模型前先处理缺失（缺失会破坏平衡评估）。
