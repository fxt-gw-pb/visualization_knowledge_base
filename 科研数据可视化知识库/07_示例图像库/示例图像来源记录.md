# 示例图像来源记录

> 全库示例图像的“来源 + license + 处置”总台账。新增示例必登记一行。处置规则见 [[资源评价标准]] 第 5 节。

## 处置规则速记

```text
license 开放（CC/BSD/MIT）→ 可下载缩略图存本地 + 注来源
license 受限 / 不明 / 期刊图 → 只记链接 + 文字，不下载
任何情况都登记：来源、作者/机构、license、抓取日期
```
