# 模型效应方向配色

> 适用：森林图、回归系数图、火山图、效应/差值图——凡是**用颜色编码“效应朝哪个方向”**的场景。
> 本质是发散语义的一个专门约定，承接 [[发散变量配色]]。

## 1. 适用场景

- 森林图/系数图：OR/HR/RR 大于 1（危险）vs 小于 1（保护）；β 正/负。
- 火山图：上调（红）vs 下调（蓝）。
- 效应差值图：正效应 vs 负效应（中心 0）。

## 2. 统一约定（registry）

| 方向 | 含义 | 颜色 | 备注 |
|---|---|---|---|
| 有害 / 正效应 / 上调 / OR>1 / HR>1 | 升高风险或上调 | `#B2182B` 深红 | 暖色=升高 |
| 无效 / 跨过参考线 / 不显著 | CI 跨 1 或跨 0 | `#999999` 灰 | 中性，弱化 |
| 保护 / 负效应 / 下调 / OR<1 / HR<1 | 降低风险或下调 | `#2166AC` 深蓝 | 冷色=降低 |

参考线：

- 比值类（OR/HR/RR）参考线 = **1**，建议 **log 轴**（见 [[森林图_ForestPlot]]）。
- 差值/系数类参考线 = **0**。
- 火山图：竖线 = ±logFC 阈值，横线 = −log10(p) 阈值。

## 3. 硬规则

1. **方向语义固定**：红=升高风险/上调，蓝=降低/下调，全文不变。
2. **“不显著”降饱和/变灰**，把注意力留给显著且方向明确的项。
3. **不要反转语义**让“有害”显得讨喜（红线，见 [[图表返修规则总表]]）。
4. 颜色只是**辅助**；效应方向的主信息靠**位置（点相对参考线）**，颜色错了也不能误导位置。
5. 比值类记得 log 轴，否则保护与有害的视觉幅度不对称。

## 4. 代码片段

**R（森林图按方向上色）**

```r
df$dir <- with(df, ifelse(low > 1, "harm",
                   ifelse(high < 1, "protect", "ns")))  # 基于 CI 是否跨 1
cols <- c(harm="#B2182B", ns="#999999", protect="#2166AC")  # >>> PARAM
ggplot(df, aes(OR, term, color = dir)) +
  geom_vline(xintercept = 1, linetype = 2, color = "grey50") +  # 参考线
  geom_pointrange(aes(xmin = low, xmax = high)) +
  scale_x_log10() +                # 比值类 log 轴
  scale_color_manual(values = cols)
```

**Python（火山图方向上色）**

```python
df["dir"] = "ns"
df.loc[(df.logFC >  1) & (df.padj < 0.05), "dir"] = "up"
df.loc[(df.logFC < -1) & (df.padj < 0.05), "dir"] = "down"
cmap = {"up":"#B2182B", "down":"#2166AC", "ns":"#999999"}   # >>> PARAM
for k,c in cmap.items():
    s = df[df.dir==k]
    plt.scatter(s.logFC, -np.log10(s.padj), c=c, s=8, label=k)
plt.axvline(1, ls="--", c="grey"); plt.axvline(-1, ls="--", c="grey")
plt.axhline(-np.log10(0.05), ls="--", c="grey")
```

## 5. 常见错误

- ❌ 不显著项也用浓色，淹没真正的信号。
- ❌ 比值类用线性轴，OR=0.5 和 OR=2 视觉不对称。
- ❌ 红蓝方向在不同图里互换。
- ❌ 颜色当主信息，忽略点相对参考线的位置才是关键。
