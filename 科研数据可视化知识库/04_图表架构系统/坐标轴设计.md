# 坐标轴设计

> 坐标轴是图里**最容易误导也最容易修**的部分。核心三问：从哪开始？能不能截断？用什么刻度（线性/log）？

## 1. y 轴是否必须从 0

| 图类型 | 从 0？ | 原因 |
|---|---|---|
| 柱状图 / 面积图 | **必须从 0** | 长度/面积编码量值，截断 = 视觉撒谎 |
| 折线图（趋势）| 不必须 | 位置编码，可聚焦数据范围；但要诚实标刻度 |
| 散点 / 箱线 / 小提琴 / 点估计 | 不必须 | 关注分布/关系，按数据范围留边距即可 |
| 比例/概率（0–1）| 视情况，常 0–1 全显 | ROC/校准固定 0–1 见各卡片 |

**铁律**：凡是用**长度或面积**编码数值的几何（bar/area），y 轴**必须含 0**。要放大差异就换 point-range / dot plot（[[点估计置信区间图_PointRange]]）。

## 2. 何时可以截断 / 放大

- 折线、点图、箱线等**非长度编码**图，可只显示数据所在区间（加合理边距，如 `expansion()`）。
- 但要**诚实**：
  - 不制造虚假差异（[[图表返修规则总表]] 红线）。
  - 若打断坐标轴（broken axis）展示极端值，**明确画出断轴符号**并标注。
- 截断 vs 全显有歧义时，给出全显版本作为补充材料。

## 3. log scale（对数轴）

何时用：

- 数据**跨多个数量级**（GLUCOSE 有极端值、表达量、p 值）。
- 比值类（OR/HR/RR）→ **log 轴让“×2”和“÷2”视觉对称**（[[森林图_ForestPlot]]）。
- 指数增长/衰减（发病、浓度）。

怎么标（必须）：

1. 在轴标题或图注**明确写 “log scale” / “log10”**。
2. 刻度用**真实值**（1, 10, 100），而非 log 后的（0,1,2）。
3. 比值图参考线 = 1；log 轴下 0.5 与 2 离 1 等距。
4. p 值常用 −log10(p)（火山图 y 轴），标注清楚。

## 4. 单位与轴标题

- 每个轴标题都带**量纲单位**：`BMI (kg/m²)`、`Systolic BP (mmHg)`、`Time (days)`、`Cholesterol (mg/dL)`。
- 量纲缩放时写清：`Expression (log2 CPM)`、`Cost (×10³ USD)`。
- 轴标题用**人类可读名**，不是变量名（`TOTCHOL` → `Total cholesterol (mg/dL)`）。
- 分类轴标签过长 → 旋转 30–45° 或换行或缩写（返修规则）。

## 5. 刻度（ticks/breaks）

- 刻度数量适中（一般 4–7 个主刻度），别太密。
- 用“好看的数”（0,5,10 或 0,25,50），避免 3.7、8.4 这种。
- 必要时次刻度（minor ticks）辅助读数，但别喧宾夺主。
- 时间轴按语义分（按访视期 1/2/3、按年/月）。

## 6. 参考线（reference line）

| 图 | 参考线 |
|---|---|
| OR/HR/RR 森林图 | x = 1（无效应）|
| 差值/系数图 | x = 0 |
| 校准曲线 | y = x 对角线（完美校准）|
| ROC | 左下到右上对角线（随机）|
| 火山图 | ±logFC 竖线、−log10(p) 横线 |
| 临床阈值 | 如 BMI=25/30、BP=140/90，用浅色虚线标注 |

参考线用**浅灰/虚线**，在数据层之下，不抢戏。

## 7. 代码片段

**R / ggplot2**

```r
+ scale_y_continuous(limits = c(0, NA),               # 柱状必含 0
                     expand = expansion(mult = c(0, .05)))
+ scale_x_log10(breaks = c(0.25,0.5,1,2,4))           # 比值 log 轴，真实刻度
+ labs(x = "Total cholesterol (mg/dL)", y = "Count")  # 带单位
+ geom_hline(yintercept = 0, linetype = 2, color = "grey60")  # 参考线
+ theme(axis.text.x = element_text(angle = 45, hjust = 1))    # 防重叠
```

**Python / matplotlib**

```python
ax.set_ylim(0, None)                       # 柱状含 0
ax.set_xscale("log"); ax.set_xticks([0.25,0.5,1,2,4]); ax.set_xticklabels([0.25,0.5,1,2,4])
ax.set_xlabel("Total cholesterol (mg/dL)")
ax.axvline(1, ls="--", c="grey")           # 参考线
plt.setp(ax.get_xticklabels(), rotation=45, ha="right")
```

## 8. 常见错误（→ 返修）

- ❌ 柱状图截断 y 轴 → 必含 0，或换 point-range。
- ❌ log 轴不标注 / 用 log 后的数当刻度。
- ❌ 轴标题缺单位。
- ❌ 标签重叠不旋转。
- ❌ 比值图用线性轴。
