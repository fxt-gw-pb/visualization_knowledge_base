# 标签与注释设计

> 注释是“给读者画重点”。原则：**只标服务结论的那几处**，宁缺毋滥。

## 1. 统计显著性标注

### p 值格式（全文统一）

```text
p < 0.001        （极小值不要写 p = 0.0000）
否则保留两位有效数字： p = 0.023 ， p = 0.0008 → p < 0.001
0.05 附近精确给值： p = 0.048 / p = 0.052（别只写 *）
```

### 星号约定（如用）

```text
ns: p ≥ 0.05    *: p < 0.05    **: p < 0.01    ***: p < 0.001    ****: p < 0.0001
```

- 星号省空间但丢失精度；论文正文图**建议给精确 p 值**，海报/汇报可用星号。
- 图注里写明星号对应阈值。

### 标多少

- **不要把所有两两 p 值都堆上**（[[图表返修规则总表]]）。
- 优先：**全局检验**（ANOVA / Kruskal-Wallis）+ 只标**预设的关键比较**。
- 多重比较要校正（Bonferroni/FDR），并在图注说明。

## 2. 误差/区间标注

- 误差棒必须**说明是什么**：95% CI / SE / SD / IQR——写在图注或图例。
- 同一文里误差度量口径统一（[[术语表]] 区分 SD/SE/CI）。
- 区间方向语义：比值类参考线=1，差值类=0（[[坐标轴设计]]）。

## 3. 重点结论注释

- 用箭头/文字标出**关键数据点或区域**（如“拐点”“显著上调基因”“高危阈值”）。
- 注释文字简短，避免长句。
- 注释颜色克制（深灰/黑），不抢数据颜色。
- 临床阈值用浅色虚线 + 小标签（BMI=25/30、BP=140/90）。

## 4. 样本量 n

- 分组图建议标各组 **n**（轴标签下、或 `n=` 标注）。
- 生存图用 **risk table** 展示各时间点 at-risk 数（[[KM生存曲线_KaplanMeier]]）。
- 过滤/缺失导致 n 变化时，在图注说明分析样本。

## 5. 数据标签（data labels）

- 柱状/点图可在末端标数值，但**别每个点都标**（拥挤）。
- 散点不标值，靠坐标读。
- 百分比堆叠图可标关键百分比。

## 6. 代码片段

**R**

```r
library(ggpubr)
# 全局 + 预设两两比较：
+ stat_compare_means(method = "kruskal.test", label.y = ymax) +              # 全局
  stat_compare_means(comparisons = list(c("A","B")), label = "p.format")     # 仅预设
# 阈值参考线 + 标注：
+ geom_hline(yintercept = 25, linetype = 2, color = "grey60") +
  annotate("text", x = Inf, y = 25, label = "BMI 25", hjust = 1.1, size = 3)
```

**Python**

```python
# 仅预设比较加 p 值（statannotations，注意 seaborn 版本兼容）
from statannotations.Annotator import Annotator
ann = Annotator(ax, [("A","B")], data=df, x="grp", y="val")
ann.configure(test="Mann-Whitney", text_format="full").apply_and_annotate()
# 阈值线 + 文字
ax.axhline(25, ls="--", c="grey"); ax.text(ax.get_xlim()[1], 25, "BMI 25", ha="right", va="bottom")
```

## 7. 常见错误（→ 返修）

- ❌ p 值满天飞 → 全局 + 预设比较。
- ❌ `p = 0.0000`；星号无图注说明。
- ❌ 误差棒不说明是 CI/SE/SD。
- ❌ 注释文字太长、太多、颜色太抢。
- ❌ 分组图不标 n。
