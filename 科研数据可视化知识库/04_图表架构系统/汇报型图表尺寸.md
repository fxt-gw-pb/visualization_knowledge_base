# 汇报型图表尺寸

> 组会/答辩 PPT 的图和论文图**目标不同**：论文是近距离精读，汇报是远距离一眼看懂。核心调整：**放大字号、提高对比、减少信息密度。**

## 1. 画布与长宽比

| 用途 | 比例 | 像素（参考）|
|---|---|---|
| 16:9 幻灯片整屏 | 16:9 | 1920×1080 |
| 幻灯片半屏（左图右文）| ~4:3 | 960×1080 |
| 海报大图 | 视海报版式 | 高 dpi 矢量优先 |

## 2. 字号 / 线宽（远看也清楚）

| 元素 | 论文 | 汇报（放大）|
|---|---|---|
| 轴标题 | 7–9 pt | **16–22 pt** |
| 刻度 | 6–8 pt | **14–18 pt** |
| 图例 | 6–8 pt | **14–18 pt** |
| 线宽 | 0.5–1 pt | **1.5–2.5 pt** |
| 点 | 小 | 放大 |

## 3. 与论文图的关键差异

1. **信息密度降一档**：一张 slide 一个结论，删次要系列/注释。
2. **对比度拉高**：深色背景幻灯可用透明背景 + 浅色文字/线（导出 PNG 透明）。
3. **颜色更大胆但仍色盲友好**（[[色盲友好与打印友好原则]]）。
4. **关键结论直接标在图上**（大字、箭头），观众没时间细读图注。
5. **同时导出 PNG + PDF**：PNG 进 PPT（≥300dpi 防糊），PDF 备矢量（[[图表返修规则总表]] 汇报规则）。

## 4. 代码片段

**R（汇报主题）**

```r
theme_talk <- theme_minimal(base_size = 18) +     # >>> PARAM base_size 大
  theme(axis.title = element_text(size = 20),
        legend.text = element_text(size = 16),
        plot.background = element_rect(fill = NA, color = NA))  # 透明
ggsave("talk_fig.png", p + theme_talk, width = 10, height = 6, dpi = 300, bg = "transparent")
```

**Python**

```python
plt.rcParams.update({"font.size": 16, "axes.titlesize": 20,
                     "axes.labelsize": 18, "lines.linewidth": 2.2})
fig, ax = plt.subplots(figsize=(10, 6))
# ... 画图 ...
fig.savefig("talk_fig.png", dpi=300, transparent=True, bbox_inches="tight")
```

## 5. 常见错误（→ 返修）

- ❌ 直接把论文小图贴进 PPT → 远处看不清，按本表放大重出。
- ❌ 一张 slide 塞满多结论 → 拆 slide / 一图一结论。
- ❌ PNG dpi 太低糊掉 → ≥300dpi。
- ❌ 深背景上用深色线 → 透明背景 + 浅色，或换白底卡片。
